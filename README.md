一款使用 **Cocos Creator** 开发的数字华容道游戏，内置 **A\*** 与 **IDA\*** 启发式搜索求解器，可自动求解任意打乱状态。

## ✨ 功能特性

- 经典数字华容道玩法（3x3 / 4x4 / 5x5 可选）
- 手动拖拽/点击移动数字块
- **自动求解**：一键展示最优解步骤（动画回放）
- 双算法支持：A* 与 IDA*
- 启发式函数：曼哈顿距离 + 线性冲突优化
- 小顶堆优先队列优化（A* 实现）

## 🧠 算法说明

求解数字华容道是典型的 **NP难问题**，本项目实现了两种主流启发式搜索算法：

- **A\***：使用曼哈顿距离作为启发函数，结合小顶堆（优先队列）保证每次扩展最优节点，适用于 3x3 规模。
- **IDA\***：迭代加深 A*，以深度 + 启发值作为剪枝依据，内存占用极低，可稳定求解 4x4 华容道。

优化细节：
- 线性冲突启发式：考虑同行/同列数字的错位阻碍，比纯曼哈顿距离更精准
- 状态去重：使用字符串编码 + Set 记录已访问状态
- 移动方向剪枝：避免“上一步→下一步”的无意义折返
